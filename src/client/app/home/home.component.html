Hello, Angular Developer!

Welcome to a blazing fast Angular Universal development starter

Features

  • FuseBox bundling (faster than Webpack and JSPM)
  • Fully typed build tools and application using TypeScript
  • Production and development builds
  • Angular testing with Jest (faster than Karma)
  • Manage your type definitions using @types (easier than using typings)
  • Simple Heroku deployment
  • Coming Soon: Cookies Support
  • Coming Soon: Auth Service + Auth Interface
  • Coming Soon: HttpStateTransfer for caching server responses on client boostrap
  • Coming Soon: Hot Module Reloading (client only dev mode)
  • Coming Soon: Support for Angular Mobile Toolkit (Service Worker)
  • Coming Soon: Ahead-of-Time (AOT) compilation support
  • Coming Soon: Tree-Shaking builds
